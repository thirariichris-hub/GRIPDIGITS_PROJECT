# GripDigits Biometric Website

A modern government-grade biometric scanning website built with:

- Next.js
- React
- Tailwind CSS
- Framer Motion
- Lucide React

## Installation

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```

Project Structure

```
public/
src/
package.json
tailwind.config.js
next.config.js
```

Features

- Responsive Design
- Modern UI
- Government Style Layout
- Animated Sections
- Dashboard Preview
- Contact Form
